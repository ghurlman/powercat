What’s New

Approvals Kit Feb release and Custom connector discussion
The February release of the Approvals Kit added the ability to provide an external reference when starting a business approval process. The external reference makes it easy to relate the approval instance to a unique identifier that you can use to update your system of record when the approval process completes or is cancelled. Want to explore the February release have a look at our LinkedIn post.

We are investigating if working with Dataverse Connector to start a business approval with the Approvals Kit will make a better experience. Want to know more your can read our LinkedIn post that explains the proposed change. Feedback from customers will help us make the case for adding this feature.
CoE Kit February release available
Our LinkedIn post provides a quick summary of the features in the February release of the CoE Kit release.

The key focus for this release was update of the kit to remove the legacy connector in response to Important changes (deprecations) coming in Power Platform - Power Platform | Microsoft Learn. By moving to the modern connector, users can take advantage of additional actions, triggers, and improvements when working with Dataverse. This change ensures that the CoE Kit remains up-to-date and fully functional within the Power Platform environment.

With this release you can now store deleted objects temporarily by default for 2 months before actual deletion.
Automation Kit Updates: Release update and roadmap
Role transition:
Marty Carreras and Power CAT Automation team are taking over the product management of the automation kit from Grant, who will focus on the CoE.

Latest release of the automation kit:
The release includes updates to the audit trail information for flow sessions, which allows for more control and visibility for administrators and support personnel.

Runtime scheduling feature:
The team is developing a feature that will enable users to define and modify the schedule of cloud and desktop flows at runtime, using a Cron scheduler mechanism and a calendar view. This will address a common pain point and add value to the Automation Kit.

Accenture
